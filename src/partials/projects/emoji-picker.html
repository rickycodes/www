Emoji Picker

Vanilla JavaScript emoji picker that replaces the basecamp emoji string, e.g. :dog: in a text input with the unicode character equivalant, e.g. üê∂. This uses native (in browser, no bundling!) supported ES6 modules, so you may need to enable that in order for it to work. A similar feature exists in the popular cloud-based team collaboration tool Slack.
